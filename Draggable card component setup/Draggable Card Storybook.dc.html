Draggable Card

A compact, draggable palette item used in the Product Sandbox component library. Grab it by the grip to drag onto a build canvas, or use the inline actions to edit or add it. Actions are fully configurable.

States

Visual states driven by the state prop. Default is also interactive — try dragging it.

Default state="default"
Hover state="hover"
Dragging state="dragging"
Selected state="selected"
Disabled state="disabled"

Configurations

Toggle the grip and each icon-button action independently. Buttons use the design-system Button (icon-only).

Edit + Add (default)
Add only showEdit={false}
Edit only showAdd={false}
No actions showAdd={false}showEdit={false}
No grip showGrip={false}
Long title truncates
Remove from selection showRemove={true}
